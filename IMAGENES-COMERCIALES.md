### 2. Extracción
```bash
# Conversión de Google Slides a PDF
curl -L "https://docs.google.com/presentation/.../export/pdf" -o salado-presentacion.pdf

# Extracción de imágenes con pdfimages
pdfimages -png salado-presentacion.pdf salado-img
```

### 3. Selección y Optimización
```bash
# Criterios de selección:
# - Tamaño original > 1MB (alta calidad)
# - Contenido relevante (no logos ni stock genérico)
# - Diversidad de categorías

# Optimización ImageMagick

```

**Resultados:**
- Formato: PNG → JPG
- Resolución: Variable → 1920px ancho
- Tamaño promedio: ~600KB (reducción ~85%)
- Calidad: 85% (óptimo para web/móvil)
